**Team Task Manager**
- A dynamic, interactive to-do list application for teams — built from scratch with  HTML, CSS, and JavaScript.

**Live Demo**
https://sanjidahjahan.github.io/Fully-Functional-To-Do-List/

**Features**
- Add teammates with duplicate and empty-input validation, sorted alphabetically
- Assign tasks to teammates with due dates and chronological ordering within each group
- Check off tasks with clear visual feedback for completed items
- "Clear Completed" removes all finished tasks and cleans up empty teammate headings
- "Reset" wipes the entire interface back to its initial state, with a confirmation prompt
- Full app state is saved to local storage and restored automatically on page reload

**Getting Started**
- Open index.html in your browser — no installs or build tools required

**Tech Stack**
- HTML
- CSS
- JavaScript 

**Project Structure**
- index.html
- IA2list.css
- IA2list.js

**What I Learned**
- This project started as a static layout and evolved into a fully interactive interface through event-driven JavaScript and careful DOM manipulation. Key challenges included designing a reliable local storage system to persist nested app state across sessions, building task and teammate flows with thorough input validation, and keeping the UI clean and consistent as data changed dynamically. It deepened my understanding of client-side architecture, DOM structure, and what it takes to build polished user interactions from scratch.
